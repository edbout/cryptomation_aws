3. _calc_optimal_edge always returns 10.0 — backtest is broken (CRITICAL)

At edge_pct=10, the threshold is 0.5 * 0.90 = 0.45. The entire backtest infrastructure — which feeds optimal_edge into the dynamic edge threshold — is producing garbage.

The correct interpretation is: "edge = how much cheaper is current market price vs. historical fair value". Use the minute's average price as the fair value proxy:


def _calc_optimal_edge(self, history: List[Dict], minute: int) -> float:
    minute_history = [h for h in history if h.get('minute') == minute]
    if len(minute_history) < 20:
        return 10.0

    minute_avg = mean(h['price'] for h in minute_history)
    if minute_avg <= 0:
        return 10.0

    best_edge = 10.0
    best_ev = 0.0

    for edge_pct in [e * 0.5 for e in range(4, 41)]:  # 2.0% to 20.0%
        # Qualify trades where price was edge_pct% below the minute's avg fair value
        qualifying = [
            h for h in minute_history
            if (minute_avg - h['price']) / minute_avg * 100 >= edge_pct
        ]
        if len(qualifying) < 10:
            continue
        wins = sum(1 for h in qualifying if h.get('direction') == h.get('outcome'))
        win_rate = wins / len(qualifying)
        avg_entry = mean(h['price'] for h in qualifying)
        ev = win_rate * (1 - avg_entry) - (1 - win_rate) * avg_entry
        if ev > best_ev:
            best_ev = ev
            best_edge = edge_pct

    return best_edge
